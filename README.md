# AllArenaAscended
Repository for my Ark Ascended Mod All Arena


Complete Control Over Boss Arena Settings!
==========================================

This mod will allow you to bring any dino into any arena, edit the teleport radius, use tek gear, during boss fights, across any map you'd like! Works in servers and single player. 

This mod is fully configurable, you can edit the parameters to whatever you like in the ini file. This will also work on the tek cave, Check down below for an ini settings guide!



Configurable settings in the GameUserSettings.ini file: 
==========================================

(Listed are the default Values of the mod) 

[AllArenaAscended] 

MaxDinos=50 (Ark's Default is 20)

MaxPlayers=50 (Ark's Default is 10)

MaxDragWeight=1000000 (Ark Default is 560)

AllowFlyers=True (This will also enable flyers in all caves, Ark's Default is False)

AllowClimbers=True (Ark's Default is False)

DefaultTeleportRadius=1440 (Ark's Default)

TekCaveTeleportRadius=4320 (Ark's Default)

AllowTek=True (Ark Default is False)

BossCooldown=600 (Ark's Default)

ExtinctionBossCooldown=21600 (Ark's Default)

ExtinctionRadius=3000 (Ark's Default)

TekCaveTime=3600

TekCaveDoorCloseTime=600

BossTimeMultiplier=1.0

LCRadius=1800

PreventDinoClasses=Empty (Example: Blueprint'/Game/PrimalEarth/Dinos/Rex/Rex_Character_BP.Rex_Character_BP' separated by commas for each entry )
